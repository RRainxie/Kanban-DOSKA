# Канбан-доска

Простая канбан-доска для управления задачами. HTML + CSS + JS, без фреймворков.

## Функции

- 5 колонок: Бэклог, В работе, Готово, Принято, Заблокировано
- Лимит задач в работе: 3
- Добавление, перемещение, удаление задач
- Сохранение в localStorage (данные не теряются после перезагрузки)

## Запуск

Открыть `kanban.html` в любом браузере.

## Как пользоваться

1. Ввести текст в поле "Новая задача" и нажать + 
2. Перемещать задачи кнопками с названиями колонок
3. Удалять задачи крестиком
4. Кнопка "Очистить всё" удаляет все задачи

## Технологии

- HTML5
- CSS3
- JavaScript (чистый, без библиотек)
